# Инструкция по настройке Google Drive для видео

## Шаг 1: Загрузка видео на Google Drive

1. Откройте [Google Drive](https://drive.google.com)
2. Создайте новую папку (например, "Royal Exchange Videos")
3. Загрузите все 11 видео файлов в эту папку:
   - `Введение.mp4`
   - `Раунд 1.mp4`
   - `Раунд 2.mp4`
   - `Раунд 3.mp4`
   - `Раунд 4.mp4`
   - `Раунд 5.mp4`
   - `Раунд 6.mp4`
   - `Раунд 7.mp4`
   - `Раунд 8.mp4`
   - `Раунд 9.mp4`
   - `Раунд 10.mp4`

## Шаг 2: Настройка публичного доступа

Для каждого файла:

1. Правой кнопкой мыши на файл → "Настроить доступ" (или "Share")
2. Нажмите "Изменить" (Change)
3. Выберите "Все, у кого есть ссылка" (Anyone with the link)
4. Убедитесь, что роль установлена как "Читатель" (Viewer)
5. Нажмите "Готово" (Done)

**Важно:** Файлы должны быть доступны для просмотра всем, у кого есть ссылка!

## Шаг 3: Получение ID файлов

Для каждого файла нужно получить его Google Drive ID:

### Способ 1: Из URL файла

1. Откройте файл в Google Drive (двойной клик)
2. Скопируйте URL из адресной строки браузера
3. URL будет выглядеть так:
   ```
   https://drive.google.com/file/d/1ABC123xyz789DEF456ghi012jkl345mno/view
   ```
4. ID файла - это часть между `/d/` и `/view`:
   - В примере выше: `1ABC123xyz789DEF456ghi012jkl345mno`

### Способ 2: Через "Получить ссылку"

1. Правой кнопкой на файл → "Получить ссылку" (Get link)
2. Скопируйте ссылку
3. Извлеките ID таким же образом

## Шаг 4: Вставка ID в код

Откройте файл `static/script.js` и найдите объект `VIDEO_DRIVE_IDS` (в начале файла, около строки 5).

Замените `'YOUR_FILE_ID'` на реальные ID для каждого файла:

```javascript
const VIDEO_DRIVE_IDS = {
    'Введение.mp4': '1ABC123xyz789DEF456ghi012jkl345mno', // Ваш реальный ID
    'Раунд 1.mp4': '2DEF456ghi012jkl345mno678pqr901stu',
    'Раунд 2.mp4': '3GHI012jkl345mno678pqr901stu234vwx',
    // ... и так далее для всех файлов
};
```

## Шаг 5: Проверка работы

1. Сохраните изменения в `static/script.js`
2. Закоммитьте и запушьте в Git:
   ```bash
   git add static/script.js
   git commit -m "Настроена поддержка Google Drive для видео"
   git push
   ```
3. Railway автоматически задеплоит обновление
4. Откройте веб-интерфейс игры и проверьте, что видео загружаются

## Важные замечания

⚠️ **Ограничения Google Drive:**
- Google Drive может ограничивать прямую загрузку больших файлов (>100MB)
- Если видео не загружаются, попробуйте использовать альтернативный формат URL
- Для файлов >100MB может потребоваться использовать Google Drive API

✅ **Альтернативный формат URL (если прямой не работает):**
Если `https://drive.google.com/uc?export=download&id=FILE_ID` не работает, попробуйте:
```javascript
const GOOGLE_DRIVE_BASE_URL = 'https://drive.google.com/file/d/';
// И в конце добавить: /preview
```

Но для этого нужно изменить код в функции `playVideo`.

## Отключение Google Drive (возврат к локальным файлам)

Если хотите вернуться к локальным файлам, измените в `static/script.js`:

```javascript
const USE_GOOGLE_DRIVE = false; // Изменить на false
```

## Проблемы и решения

**Проблема:** Видео не загружаются
- Проверьте, что файлы публичные (доступны всем по ссылке)
- Проверьте, что ID файлов правильные
- Проверьте консоль браузера (F12) на наличие ошибок

**Проблема:** Видео загружается, но не проигрывается
- Google Drive может блокировать прямую загрузку больших файлов
- Попробуйте использовать альтернативный формат URL (см. выше)

**Проблема:** Ошибка CORS
- Google Drive может блокировать запросы с других доменов
- В этом случае лучше использовать другой хостинг (Cloudflare R2, GitHub Releases)
